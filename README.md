# Censorship

## Features
+ Chat observer
    + Observes chat
    + "Blocks" incompatible words
+ Word list
    + Can be inserted into chat observer to determine which words are blocked
    + Written by the player
+ VPN
    + Counteracts the effects of the chat observer
    + Totem-like (single use)
